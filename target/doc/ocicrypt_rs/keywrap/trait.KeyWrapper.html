Expand description

KeyWrapper is the interface used for wrapping keys using a specific encryption technology (pgp, jwe, pkcs7, pkcs11, keyprovider)

Provided Methods§

source

fn private_keys( &self, dc_param: &HashMap<String, Vec<Vec<u8>>> ) -> Option<Vec<Vec<u8>>>

private_keys (optional) gets the array of private keys. It is an optional implementation as in some key services, a private key may not be exportable (i.e. HSM) If not implemented, return None.

source

fn keyids_from_packet(&self, packet: String) -> Option<Vec<u64>>

keyids_from_packet (optional) gets a list of key IDs. This is optional as some encryption schemes may not have a notion of key IDs If not implemented, return None.

source

fn recipients(&self, recipients: String) -> Option<Vec<String>>

recipients (optional) gets a list of recipients. It is optional due to the validity of recipients in a particular encryption scheme If not implemented, return None.
